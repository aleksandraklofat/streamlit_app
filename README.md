### calling streamlit app (Befehle)
On Anaconda: 
1. Open Anaconda Navigator
2. Environments
3. Run Environment
4. Open Terminal
5. cd (richtiges Directory)
6. "streamlit run app.py"

On Terminal (with VS Code)
1. Open "Terminal" in VS Code
2. run in Terinal "# python -m streamlit run app.py"
